# 概要

[「瑠璃の泪」ティザーサイト](https://tear-of-lapislazuli.herokuapp.com/)のソースコードになります。

# ライセンス
MITライセンスとして公開しています。

# 開発環境

Ruby 2.3.3

Ruby on Rails 5.1.4

# 本番環境

Herokuを使用し、Github上のソースコードにコミットが確認された段階で自動的に最新版をデプロイするようにしてあります。
そのため、挙動が不安定になることもあるかと思います。
あらかじめご了承ください。
